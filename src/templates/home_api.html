Bienvenue sur notre API



Cette API Contient une analyse sur les movements des candidats en temps réels pendant leur campagne politique (avant les elections) et meme pendant les elections. Il sert à analyser leurs mouvements et deplacement et predire les actions qu'ils auront à mener à partie d'une dashboard de leurs actions precedente ; de scruter/scraper média ou les reseaux sociaux et en deceler le candidat le plus apprécié, celui vers qui vont se tourner la majorité des votes ; Enfin, il analyse la tendance quotidienne des votes.
Pour plus d'information, rendez-vous à l'adresse suivante : documentation-API-data-analysis-and-electoral-movements, mis à jour le 22 Avril 2021
